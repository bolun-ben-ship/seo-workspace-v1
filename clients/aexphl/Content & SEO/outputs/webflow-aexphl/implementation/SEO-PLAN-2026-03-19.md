## 🔴 Critical — Fix This Week

### 1. Noindex `/landing-page` and `/landing-page-v2`
**Why:** Both pages are indexed and competing with the homepage for branded and core keywords. Landing pages built for paid ads should never be indexed.
**How:** In Webflow → Page Settings → SEO tab → uncheck "Index this page" for both pages. Also add `<meta name="robots" content="noindex, nofollow">` via the custom head code embed as a double-lock.
**Impact:** Consolidates link equity to homepage, removes cannibalisation risk.
**Effort:** 10 minutes.

### 2. Fix Brand Name Fragmentation in Title Tags
**Why:** Google sees "AEHL", "Aussie Expat Home Loans", and "AEXPHL" as potentially separate entities, diluting entity authority.
**How:** Standardise all title tags to use "Aussie Expat Home Loans" as the brand suffix. Never use "AEHL" in a title tag — it's an internal acronym with zero search value.
**Effort:** 30 minutes across all pages.

### 3. Rewrite All Inner Page Title Tags

**Effort:** 30 minutes in Webflow.

### 4. Write Meta Descriptions for All Pages
Google is auto-generating snippets because meta descriptions are missing or weak. Craft one compelling 150-160 character meta description per page. For financial services, lead with the outcome (e.g., *"Get your Australian borrowing capacity assessed in 48 hours — specialist expat mortgage broker with offices in Singapore, Hong Kong and Melbourne."*)
**Effort:** 1–2 hours for all core pages.

### 5. Add FAQ Schema to `/faqs`
**Why:** FAQ schema is the fastest path to a rich result in SERP. An expat searching "can I get an Australian mortgage while living overseas?" could see AEXPHL's answer directly in Google.
**How:** Add `FAQPage` JSON-LD to the `/faqs` page custom code embed in Webflow.

```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "Can I get an Australian home loan as an expat?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "Yes. Australian citizens and permanent residents living overseas can apply for Australian home loans. Lenders assess foreign income under specific expat policies. [Continue answer...]"
      }
    }
  ]
}
```
**Effort:** 2–3 hours (write questions + implement).

### 6. Add Organization + LocalBusiness Schema to Homepage
**Why:** Establishes AEXPHL as a known entity in Google's knowledge graph, enables multi-location business features, improves AI search citation likelihood.
**How:** Add JSON-LD to homepage custom head code in Webflow. Include: `legalName`, `url`, `logo`, `sameAs` (LinkedIn, Facebook), `contactPoint` for all three offices, `areaServed`.
**Effort:** 1 hour.

### 7. Create Singapore Location Landing Page
**Why:** Odin Mortgage has a dedicated `/australian-mortgage-singapore/` page already capturing this traffic. Singapore is AEXPHL's primary market. This is the single highest-ROI page you can create.
**Target keyword:** "Australian home loan Singapore" / "Australian expat mortgage Singapore"
**Page brief:**
- H1: "Australian Home Loans for Expats in Singapore"
- Cover: SGD income assessment, FIRB (not applicable for citizens), lender policy for Singapore residents, CPF considerations, timeline
- Include: office address in Singapore, local phone number, trust signals
- CTA: Book consultation or borrowing capacity check
**Effort:** 3–5 hours to write + build.

## 🟡 Medium — Fix Within 1 Month

### 8. Create Hong Kong and Dubai Location Pages
Same template as Singapore (#7). Three pages, three markets, three distinct organic traffic streams.
- `/home-loans-hong-kong` — target "Australian home loan Hong Kong expat"
- `/home-loans-dubai` — target "Australian expat mortgage UAE" / "home loan Australia Dubai"

### 9. Add Article + Author Schema to All Blog Posts
**Why:** Google's E-E-A-T guidelines for YMYL (financial) content require demonstrated expertise and authorship. Without author schema, blog posts have no structured trust signal.
**How:** Add `Article` + `Person` (author) JSON-LD to every blog post. Author profiles should reference real team members (Tim, Shaun, Charu) with credentials.

### 10. Audit and Redirect or Rewrite Generic Blog Posts
These posts are fighting for keywords dominated by major banks and aggregators — unwinnable at AEXPHL's current domain authority:
- "Home Loans Australia: Options, Requirements and Comparisons"
- "How to Prepare the Minimum Deposit for a Home Loan in Australia"
- "Housing Interest Rate in Australia: A Quick Guide"
- "How to Get The Cheapest Home Loan Interest Rates in Australia"

**Options:**
- Rewrite with expat angle (e.g., "Minimum Deposit for Australian Home Loans as an Expat") to target an achievable long-tail
- Or consolidate weak posts into a single stronger guide and 301 redirect old URLs

### 11. Add Internal Links from Blog to Conversion Pages
Every blog post should contain at least one contextual link to `/book-appointment` or the stamp duty calculator where relevant. Use keyword-rich anchor text (e.g., "check your borrowing capacity" → links to book appointment).

### 12. Add Breadcrumb Schema Sitewide
Improves SERP appearance and signals site structure to Google. Easy one-time Webflow implementation.

### 13. Add Content to Stamp Duty Calculator Page
Calculator pages rank on surrounding content, not the tool alone. Add 400–600 words explaining: what stamp duty is, how it differs by state, what expats need to know about stamp duty (FIRB surcharges, foreign purchaser duty). This transforms the calculator from a thin utility page into a rankable resource.
